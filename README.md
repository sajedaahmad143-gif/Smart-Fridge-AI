# 🧊 Smart Fridge AI

## 📌 Project Idea
A smart system that helps reduce food waste by tracking expiration dates and suggesting recipes.

## ⚙️ Features
- Add food items
- Track expiration automatically
- Alert for near-expiry items
- Suggest meals using AI logic

## 🤖 AI Concept
The system uses rule-based AI to analyze food items and generate smart recipe suggestions.

## ☁️ Cloud Usage
Developed using Antigravity IDE (cloud-based environment).

## 🚀 Future Improvements
- Integration with Google Cloud
- Use Gemini AI for dynamic recipe generation
